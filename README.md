# 🛒 E-Commerce Sales Analysis

## 📌 Project Overview
This project was completed as part of a Data Analytics Internship Project Simulation.  
The objective of this project is to analyze e-commerce sales data and generate meaningful insights to support business decision-making.

---

## 🎯 Objectives
- Perform data cleaning and preprocessing  
- Analyze sales trends and patterns  
- Detect outliers in key variables  
- Visualize data using graphs and charts  
- Generate data-driven business insights  

---

## 🛠 Tools & Technologies Used
- Python  
- Pandas  
- Matplotlib  
- Google Colab  

---

## 📊 Analysis Performed
- Data cleaning and handling missing values  
- Outlier detection using box plots  
- Sales and revenue trend analysis  
- Discount impact on sales  
- Visualization of key metrics  

---

## 📈 Key Insights
- Identified unusual patterns in pricing and revenue  
- Observed relationship between discounts and sales  
- Detected outliers affecting overall analysis  
- Derived insights useful for improving sales strategies  

---

## 📁 Project Files
- `intern_1.ipynb` → Python analysis notebook  
- `INTERNSHIP_DATASET.csv` → Dataset used for analysis  
- `E-Commerce Sales Optimization.` → Project report (format as uploaded)  
- `Shumaila_Fathima_Data_Analytics_Internship_Certificate.pdf` → Internship certificate  

---

## 🚀 How to Run the Project
1. Open the notebook in Google Colab or Jupyter Notebook  
2. Upload the dataset (`INTERNSHIP_DATASET.csv`)  
3. Run all cells to reproduce the analysis
